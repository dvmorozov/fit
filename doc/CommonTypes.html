TFitServerState = (...);

Server states. Sequence of states is designated by numbers.

Values
  • ProfileWaiting: waiting of loading profile data (1)
  • BackNotRemoved: background isn't removed yet after last profile loading; state must not change on loading background points (2)
  • AsyncOperation: long operation is performed (4)
  • ReadyForAutoFit: background removed (ready to fit parameters in automatic mode) (3)
  • ReadyForFit: ready to fit with given user constraints (3)
