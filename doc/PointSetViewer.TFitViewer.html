Unit

PointSetViewer

Declaration

type TFitViewer = class(TComponent)

Description

Component which can show client data by means of UI components. Supports correspondence between series of Chart and items of CheckListBox. On turning off CheckListBox item serie can be only hidden but not deleted because there is no direct connection between items of CheckListBox and items of PointsSetList.
