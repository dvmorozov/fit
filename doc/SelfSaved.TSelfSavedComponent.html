Unit

SelfSaved

Declaration

type TSelfSavedComponent = class(TComponent)

Description

Component which can serialize itself into stream allowing adding new attributes and changing order of inheritance without loosing ability to read existing files (backward compatibility). Writing (reading) class attributes starts from attributes of the class which is last in the inheritance chain. Reading / writing operations are implemented as class operations to provide possibility of moving by inheritance chain.
